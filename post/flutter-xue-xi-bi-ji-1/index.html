· 2019-06-27 ·

Flutter是什么?

Flutter是谷歌的开源的垮平台(不限于Android和Ios)的移动应用框架。有以下特性：响应式的框架，高速的2D渲染引擎。

Flutter为什么选择dart语言?

Dart是一个简洁的、强类型的面向对象的编程语言。 性能，包含了两种编译模式Just-In-Time(即时编译)和Ahead-Of-Time(运行前编译)。JIT使Flutter在应用运行时同时执行代码编译，这就让Flutter具备了极速的开发体验，具备了亚秒级的热重载(Hot Reloading)特性。AOT运行前编译意思是将代码库直接编译成原生的ARM指令集，这样就使得App有了快速的启动速度和可预见的卓越性能。Dart同时具有强类型和类型推导的特性。这样可以让bug暴露于编译期间，更放心地应付代码量的增长。

Widget是Flutter应用的基础单元

StatelessWidget:无状态的控件，这意味着该Widget一旦创建，它所有属性都是最终状态。在widget创建的时候我们可以传递参数 比如背景颜色来设置widget的属性，但是一旦创建好之后，这些属性是不可变的。

StatefulWidget:有状态的控件，它们可以创建State对象。

Flutter的2d渲染引擎使用的是Skia，它提供了再ios和Android平台绘制Widget的能力

	//

  Widget build(BuildContext context) {
	 // 创建Center控件，这其实一个类，在构造方法内传入属性，child。
    return Center(
		// 同理，创建Text控件，传递两个属性，一个是string，另外一个是对象。
      child: Text(
        'Hello World!',
        textDirection: TextDirection.ltr,
      ),
    );
  }
}
