安卓开发入手JavaWeb踩坑之旅（一）

· 2019-07-12 ·

1.Java web开发，现在流行什么技术？

在boss直聘上搜java职位，总结出大概一下要求的关键字 关键字 spring，spring mvc，mybatis
ioc、aop、jvm radis mysql

2.开发工具

通过调研，开发工具一般选择IDEA(用IntelliJ IDEA ) + maven 进行构建项目 参考链接：开发工具参考

Maven是一个远程的第三方jar库，下好idea后，在Setting中，搜索"maven"，可以进行配置，我使用的是idea自带的配置，bundled(Maven 3)

3.构建一个项目

我使用的IDEA版本为2019.1.3

  • 1.File -》 New -》 Project

  • 2选择Maven，勾选 create from archetype，选中.......-archetypetype-webapp

  • 3 填入groupId和ArtifactId，点击Next

  • 4 出现如下图所示

  • 5 然后点击Finish，接着会Build一段时间，出现BUILD SUCCESS 的提示。

  • 6.配置Tomcat(需要下载好，我下载的版本是apache-tomcat-9.0.22)，点击Add Configuration。

  • 7.点击"+"号，找到Tomact Server，选择Local，如果找不到Tomact Server，需要往下拉，选择33 more items，因为idea只显示的部分可以添加的选项，选择下载好的解压后的(bin的上一级)目录，之后选择development,点击加号，选择Artifacts，然后选择...war explored.点击apply

  • 8这个时候，tomact已经配置好，点击启动按钮即可启动

  • 9创建java源文件目录，选中main目录，右键，New -》 Directory 输入java，然后，选中File-》Project Structure，在新建的java目录下右键，Sources即可。

  • 10 解决tomcat乱码，启动后，发现IDEA console，启动服务器乱码，经过搜索，解决方案如下:

// 1.修改apache-tomcat-9.0.14-windows-x64/conf/logging.properties
//2、添加语句：java.util.logging.ConsoleHandler.encoding = GBK
//3、重启tomcat，查看日志数据即可
