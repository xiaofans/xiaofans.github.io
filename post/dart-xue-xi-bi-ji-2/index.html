Dart学习笔记2

· 2019-05-27 ·
  • 所有的函数都有一个返回值，对于单行(Single-line shorthand)方法，这个返回值是单行表达式的结果。除此之外，函数的返回值默认是null直到你定义了一个返回值。
  • 如果需要，你可以在函数名字前定义你需要的返回类型，这个返回类型在Dart中是可选的，如果不定义，那么会返回一个dynamic类型的结果。当然，也可以使用void来表示函数并没有一个返回类型。
pls(int a,int b){
  return a+b;
}

// pls_1 == pls_2
dynamic pls_1(int a,int b){
  return a + b;
}

int pls_2(int a,int b){
  return a + b;
}
  • 可选位置参数，使用中括号标识可选参数
// 使用中括号来标识可选参数
a(int a1,[int a2,int a3]){

}


// 可选参数，赋以默认值
b(int b1,[int b2 = 1,int b3 = 2]){

}
  • 可选命名参数，与可选位置参数的区别是，调用的时候，参数名必须被指定
// 可选命名参数，使用大括号标识，参数的默认值的赋值使用冒号
b(int b1,{int b2 : 1,int b3 : 2}){

  print(b1+b2+b3);
}

//调用
b(1,b2: 1,b3: 1);
