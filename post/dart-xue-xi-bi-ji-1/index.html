Dart学习笔记1

· 2019-05-22 ·

Dart是什么? Dart是一个开源的、结构化的用于快速地构建复杂的基于浏览器的web程序。Dart开源通过转换成js在浏览器上运行。 它的语法和Java、C#、Javascript非常类似。

void main(){   // main 函数 入口
  var d = "Dart"; //  可以未指定声明类型
  String w = "World"; // 指定声明类型
  
  print("Hello $d $w");
}

Dart 与java的不同

  • 在Dart中，所有的都是对象，包括方法(functions),这就意味着你可以保存一个方法到变量。
  • shorthand标记=>，可以被用于仅有一行的函数体，
String sayHello(name) => "Hello $name";
  • top-level functions，不需要一定定义类，dart可以直接运行顶层方法。
  • 使用类结构定义接口
  • 工厂构造方法提供默认实现。

void topLevelFun(){
  print("I am form top level");
}

abstract class IGreetable{
  String sayHello(String name);

  factory IGreetable(){
    return new Greeter();
  }
}

class Greeter implements IGreetable{
  @override
  String sayHello(String name) {
    return "Hello $name";
  }
}

void main(){
  topLevelFun();
  IGreetable myGreetable = new IGreetable();
  var message = myGreetable.sayHello("Dart");
  print(message);
}
  • 方法可以作为参数传递、赋值，调用。
String sayHello(name) => "Hello $name";

void main(){

  var myFunc = sayHello;
  print(myFunc("World"));

  var  mySumFunc = (a,b){
    return a+b;
  };// 声明匿名方法

  var c = mySumFunc(1,2);
  print(c);
}
  • 并发与 isolates*[ˈaɪsəleɪts] iseleici*

Dart的执行与Java类似，是单线程语言，不同于线程和进程,Dart通过isolates实现并发，isolates是一个工作单元，有自己的内存分配空间，这样有助于提供一个独立的安全模型， 每一个isolated可以通过传递消息(messages)给别的isolates。

isolates还可以被用来动态的加载额外的代码，这段代码可以在加载带isolates的独立内存中运行，通过传递消息进行沟通，这个功能可以用于插件机制。

当isolates转化成为JavaScript，它们就变成了HTML5的works。
