C语言学习笔记

· 2019-05-14 ·

程序设计与C语言

  • 1.世界语言排名网站 https://www.tiobe.com/tiobe-index/
  • 2.C语言编程软件IDE Dev C++
  • c语言是一种编译型的语言

计算

  • 表达式的概念 "="是赋值运算符，有运算符的式子就叫做表达式。
  • const修饰符代表常量，不可变变量。
  • float和double:float单精度浮点数，double双精度浮点数。
  • 运算符(operator)是指进行运算的动作，比如加法运算符"+"，减法运算符"-"。
  • 算子(operand)是指参与运算的值，这个值可能是常数，也可能是变量，还可能是一个方法的返回值。
  • 单目运算符概念:只有一个算子的运算符：+、-
  • 赋值运算符：赋值也是 运算，也有结果。
  • 程序是按步执行的。
  • count++代表给count加1，表达式的值是count原来的值
  • ++count给count加1，表达式的值是count+1以后的值

数据类型

  • unsigned:在整形类型前面加unsigned使得它们成为无符号的整数(没有负数，都是整数)，unsigned的初衷并非扩展所能表达的范围，为是为了做纯二进制运算，主要是为了移位。
  • 8进制和16进制，一个以0开始的数字字面量是8进制，一个以0x开始的数字字面量是16进制，%o用于8进制，%x用于16进制。

指针

  • 指针前面的※是一个单目运算符，用来访问指针的值所表示的地址上的变量，可以做右值也可以做左值 int k = *p; *p = k+1;
  • 指针的运用场景:1.交换两个变量的值 2.函数返回多个值，某些值就只能通过指针返回
  • 指针最常见的错误:定义了指针变量，还没有指向任何变量，就开始使用指针。
  • 数组变量是特殊的指针，所以 int a[10]; int *p = a;但是数组的单元表达的是变量，需要用&取地址,,a==&a[0]。运算符[]可以对数组做，也可以对指针做p[0] a[0]。数组变量是const的指针，所以不能被赋值。

结构类型

Typedef:声明新的类型的名字,新的名字是某种类型的别名，改善了程序的可读性。Typedef int Length; // Length就等价于int类型。

程序结构

  • #开头的是编译预处理指令，它们不是c语言的成分，但是c语言离不开它们
  • #define用来定义一个宏，#define <名字> <值>，没有分号结尾，因为不是C的语句，名字必须是一个单词，值可以是各种东西。在C语言的编译器开是编译之前，编译预处理程序(cpp)会把程序中的名字换成值。完全的文本替换。

大程序结构

  • 头文件，把函数的原型放到一个头文件(以.h结尾)中，在需要调用这个函数的源代码文件(.c)中#inclue这个头文件，就能让编译器在编译的时候知道函数的原型。
  • #include是一个编译预处理指令，和宏一样，在编译之前就处理了
  • 它把那个文件的全部文本内容原封不动地插入到它所在的地方
  • 所以也不是一定要在.c文件的最前面#include
  • include "a.h" 编译器在当前目录(.c文件所在的目录)寻找这个文件，如果没有，到编译器指定的目录去找。include <a.h> 让编译器只在指定的目录去找，编译器知道自己的标准库的头文件在哪里。
